# 📘 Today I Learned

### 1. 오늘 배운 내용
- List 와 Map 과 같은 자료구조를 사용했다.
Map 을 활용하여 파트별로 그룹화했다.
반복문에서는
for (int i = 0; i < members.size(); i++) {
    Role member = members.get(i);
}
이런 방식처럼 인덱스를 사용할 수도 있고
for (Role member : members) {}
이런 방식처럼 더 간단하게 순회할 수 있다.
함수를 사용하여 기능별로 코드를 분리했다.

### 2. 핵심 정리 (내 언어로)
- Map은 key-value 형태로 데이터를 저장하고, 실습에서는 같은 파트끼리 그룹화할 때 사용했다.
함수를 사용해서 기능을 분리하면 코드가 더 구조적으로 보인다.

### 4. 느낀 점
- List와 Map의 차이를 이해할 수 있었다.
많은 기능을 함수로 나누어 분리하여 관리하기 편리함을 느꼈다.
멤버 등록 시에 List에만 추가하고, Map에는 추가하지 않아 오류가 발생했었다. 자료구조에 데이터를 추가해야 한다.
